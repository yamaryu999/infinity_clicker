# 🚀 無限クリッカー - Idle Clicker Game

中毒性のある無限クリッカーゲームです！シンプルなクリックから始まり、アップグレードを重ねて自動化していく放置ゲームです。

## 🎮 ゲームの特徴

### 基本機能
- **クリックでポイント獲得**: メインボタンをクリックしてポイントを稼ぎます
- **自動クリッカー**: アップグレードで自動的にポイントを生成
- **クリック倍率**: クリックごとのポイントを増加
- **クリティカルヒット**: ランダムでポイントが3倍に！

### アップグレードシステム
- 🤖 **自動クリッカー**: 自動的にポイントを生成
- ⚡ **クリック倍率**: クリックごとのポイントを増加
- 🚀 **自動クリッカー速度**: 自動クリッカーの速度を向上
- 💥 **クリティカルクリック**: クリティカルヒットの確率を向上

### 実績システム
- 様々な条件を達成すると実績が解除されます
- 実績解除時には特別な通知が表示されます

### その他の機能
- 💾 **自動セーブ**: ゲーム進行が自動的に保存されます
- 📱 **レスポンシブデザイン**: スマートフォンでも快適にプレイ
- 🎨 **美しいUI**: モダンで美しいデザイン
- ⌨️ **キーボードショートカット**: スペースキーでクリック

## 🚀 セットアップ方法

### 必要なもの
- モダンブラウザ（Chrome、Firefox、Safari、Edge等）

### インストール手順
1. このリポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く
3. ゲーム開始！

```bash
# リポジトリをクローン
git clone [repository-url]
cd webgame

# ブラウザで開く
open index.html  # macOS
# または
start index.html  # Windows
# または
xdg-open index.html  # Linux
```

## 🎯 ゲームの遊び方

1. **クリック開始**: メインボタンをクリックしてポイントを稼ぎます
2. **アップグレード購入**: 十分なポイントがたまったらアップグレードを購入
3. **自動化**: 自動クリッカーを購入して放置でもポイントを稼ぐ
4. **効率化**: 様々なアップグレードを組み合わせて効率を最大化
5. **実績解除**: 様々な条件を達成して実績をコンプリート

## 🔧 カスタマイズ

ゲームのバランス調整や新機能の追加が簡単にできます：

- `script.js` の `upgradeCosts` でアップグレードコストを調整
- `achievements` 配列に新しい実績を追加
- `style.css` でデザインをカスタマイズ
